# 🩸 LifeLink – Blood & Organ Donation Management System

LifeLink, the 3rd place for Project Excellence Award in Computer Science Capstone Projects at AlMaaref University, is a smart web platform created to connect blood, organ, and financial supporters with hospitals that need help. 

LifeLink makes the donation process easy, fast, and friendly by using modern technology. Donors can create profiles, track their donation history, get updates, and book home appointments. Hospitals can manage requests and plan schedules. The system also includes a chatbot to guide users, a human body visual to educate donors, and a reward system to keep people motivated. 

This project was developed as a capstone graduation project following complete Software Engineering lifecycle practices, including system analysis, UML modeling, database design, implementation, and deployment.

---

## 🎯 Project Objectives

- Enable seamless registration and communication for donors and hospitals
- Educate users through an interactive human body visual, allowing them to explore and understand which organ s and body parts can be donated
- Provide 24/7 assistance through a smart chatbot
- Promote recurring donations through gamification and education
- Ensure data security, privacy and legal compliance in all operations

## 🏗 System Architecture

The system follows a client-server architecture:

- **Frontend (Web Application)** – User interaction and dashboards  
- **Mobile Application (React Native)** – Cross-platform donor interaction  
- **Backend API (Laravel)** – Business logic and authentication  
- **Database (MySQL)** – Data storage and relational modeling  

---

## 🗃 Database Design

The database was designed following normalization best practices.

Main entities include:

- Users (Donors, Mobile Phlebotomists, Admins, Hospitals)  
- Blood Requests  
- Organ Requests 
- Surgical Fundings 
- Donations  
- Medical Cases  
- Gamification 
- Reward System (Xps)
